# 숟가락 대장간 — Product Brief

## 한줄 소개
돌덩이를 두드려 수저를 만들고, 모은 수저를 매일 토스포인트로 교환하는 캐주얼 리워드 앱

## 타겟 사용자
- 주 타겟: 토스 앱을 일상적으로 사용하는 20-40대 사용자
- 사용 맥락: 하루 2-5분, 매일 반복 방문 (데일리 리워드 루틴)
- 반복 사용 동기: 매일 무료 2회 기회 + 누적 수저 교환 + 스트릭 유지

## 앱 타입
- [x] Non-Game WebView (Next.js 14 + Tailwind CSS)
- [ ] Game WebView (Vite + Vanilla JS)

> 근거: 포인트 교환 화면, 보관함 등 카드 기반 데이터 UI가 핵심. framer-motion으로 탭 애니메이션 충분히 구현 가능.

## 핵심 게임 루프

### 1회 시도 (Attempt) 흐름
1. 메인 화면에서 "바위 두드리기" 버튼 탭
2. **Pre-roll**: 시도 시작 시 수저 종류 즉시 추첨 (사용자에게 비공개)
3. 추첨된 수저 등급에 따라 목표 탭 수 결정
4. 사용자가 목표 탭 수만큼 화면을 탭 → 돌 균열 진행
5. 탭 완료 → 수저 등장 연출 → 보관함 자동 추가

### 수저 티어 테이블
| 등급 | 포인트 가치 | 드롭 확률 | 필요 탭수 |
|------|------------|-----------|-----------|
| 💎 다이아 수저 | 1,000원 | 0.001% | 10,000번 |
| 🥇 금수저 | 10원 | 10% | 100번 |
| 🥈 은수저 | 5원 | 20% | 50번 |
| 🥉 동수저 | 3원 | 30% | 30번 |
| 🪨 돌수저 | 1원 | 40% | 10번 |

> 탭 방식: 실제 탭 (자동 진행 없음). 다이아 수저는 확률 0.001%로 사실상 잭팟 — 해당 시 10,000번은 플레이어가 직접 탭.
> 확률 합계: 0.001% + 10% + 20% + 30% + 40% ≈ 100% (다이아 델타 무시)

### 일일 제한
- **무료 기회**: 2회/일 (KST 자정 리셋)
- **광고 추가 기회**: 최대 3회 추가 → 하루 최대 5회
- 광고 없이도 2회 핵심 기능 사용 가능 (광고 강제 없음)

## 수익화 전략

### 광고 플레이스먼트 (3개, 모두 보상형 rewarded)

| 광고 | 트리거 | 보상 | 빈도 |
|------|--------|------|------|
| **AD-1** 기회 추가 | 메인 화면 "기회 추가" 버튼 | 시도 1회 추가 (일 최대 5회) | 일 최대 3회 |
| **AD-2** 해머 강화 | 메인 화면 "해머 강화" 버튼 | 당일 전체 탭수 절반 (자정 리셋) | 일 1회 |
| **AD-3** 다이아 부스터 | 메인 화면 "부스터" 버튼 | 1시간 다이아 확률 ×10 (0.001% → 0.01%) | 일 1회 |

### 광고 UX 원칙
- 광고는 사용자가 "더 하고 싶다"는 순간에 자연스럽게 제안
- AD-1: 기회 소진 후 안내 → 부족함 해소 동기
- AD-2/3: 메인 화면 상시 노출 → 효율 극대화 욕구 자극
- 세션당 평균 광고 노출 예상: 1-2회 (보수적 추정)

### AdMob 설정
- **개발/테스트**: `ait-ad-test-rewarded-id`
- **프로덕션**: 앱인토스 콘솔 제출 직전 교체 (별도 QA 확인 필요)

## 포인트 교환 (v1 플레이스홀더)

### 교환 규칙
- **교환 조건**: 보관함 누적 가치 **10원 이상**이어야 교환 버튼 활성화
- **교환 횟수 제한**: 하루 1회 (KST 자정 리셋)
- **교환 후 보관함**: 교환한 수저만 차감, 나머지 잔존
- **v1 구현**: UI 플레이스홀더 (교환 완료 연출만, 실제 포인트 미지급)
- **v2 목표**: 앱인토스 토스포인트 API 실연동 (심사 통과 후)

> DB 미연동 결정: 앱인토스 심사 전 실제 포인트 지급 불가. 어뷰징 방지는 실제 연동 시점에 Vercel Serverless + Upstash Redis로 추가.

## 핵심 기능 (MVP)

### 필수
1. **메인 화면** — 돌덩이, 남은 기회 표시, 하단 수저 요약
2. **탭 인터랙션** — 실제 탭 → 균열 애니메이션 → 수저 획득
3. **수저 보관함** — 누적 수저 목록 + 오늘 합산 가치
4. **포인트 교환 UI** — 플레이스홀더 교환 흐름
5. **광고 오퍼 시트** — 3개 광고 플레이스먼트

### 있으면 좋음
6. **스트릭 카운터** — 연속 방문일 수 표시 (리텐션 강화)
7. **전체 수집 통계** — 등급별 누적 수집 수 (트로피 느낌)

### v2 이후
- 실제 토스포인트 API 연동
- 소셜 공유 (금수저 뽑았을 때 공유)
- 리더보드
- 다이아 수저 특별 연출 (10,000번 달성 시 폭죽)
